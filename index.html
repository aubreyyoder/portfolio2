Show Me the Money :

This is a currency conversion app developed to allow users to convert money from one currency to another. The user inputs their own amount to convert. When given the converted amount, the user has the option to start over or look up a place to convert their money. Using multiple API's, the user converts and searches for banks nearby their given location.
